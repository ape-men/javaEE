# 判断邮箱

## 1.接口描述

接口请求域名:http://www.bluemsun.work:8080/register/checkoutEmail
注册时判断邮箱是否已使用
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
|  email  |  是   | String | 邮箱 |

## 3.输出参数

| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
| success | Number | 用户名是否已存在 |

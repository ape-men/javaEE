# 个人页

## 1.接口描述

接口请求域名:http://www.bluemsun.work:8080/personPage/psersonInfo
列出所有个人资料
请求方式:get

## 2.输入参数

无

## 3.输出参数

| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
| success | Number |  获取个人资料是否成功  |
| userName | String |    用户名        |
|  email  | String  |   电子邮箱   |
|  status  |  Number  |  用户身份      |
